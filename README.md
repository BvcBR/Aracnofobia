# ğŸš€ How to Run the Project
This project is stored as a .gmz file, which is a compressed project package from older versions of GameMaker (GameMaker: Studio 1.4).

To open and play this:

Download GameMaker: You will need GameMaker Studio 2 (it is backward compatible) or the legacy GameMaker: Studio 1.4 if you have access to it.

Import the Project:

Open GameMaker.

Select Import.

Choose the Arachnophobia.gmz file.

Note: It will ask to convert the project to the new format. This is normal.

Run: Press F5 or click the "Play" button to compile and launch the game.
